# 智知（ZhiZhi）—— AI 驱动的个人知识管理系统

## 快速启动

### 第一步：配置环境（只需一次）

#### 1. MySQL - 执行建表 SQL
```sql
-- 在 MySQL 8 中执行 sql/zhizhi.sql
source D:/U盘文件/项目代码/Zhizhi/sql/zhizhi.sql
```

#### 2. Redis - 启动服务
```bash
# Windows 下载 Redis 并运行
redis-server
```

#### 3. ChromaDB - Docker 启动
```bash
docker run -d -p 8000:8000 --name chromadb chromadb/chroma
```

#### 4. Ollama - 本地 Embedding（免费）
```bash
# 1. 安装 Ollama：https://ollama.com/download
# 2. 拉取 Embedding 模型（约 274 MB）
ollama pull nomic-embed-text
# 3. Ollama 默认在后台运行（访问 http://localhost:11434 验证）
```

#### 5. 修改后端配置
编辑 `zhizhi-backend/src/main/resources/application.yml`：
- 修改 MySQL 密码：`password: 你的密码`
- 填入 DeepSeek API Key：`api-key: sk-你的Key`（去 platform.deepseek.com 申请，充 ¥10 够用）

---

### 第二步：启动后端

```bash
cd zhizhi-backend
mvn spring-boot:run
```
访问 http://localhost:8080/api/doc.html 查看接口文档

---

### 第三步：启动前端

```bash
cd zhizhi-frontend
npm install
npm run dev
```
访问 http://localhost:3000

## 使用流程

1. **注册/登录** → 进入知识库
2. **创建知识条目** → 填写标题和 Markdown 正文
3. **向量化** → 在知识详情页点击「向量化」按钮（或批量向量化）
4. **AI 问答** → 在「AI 问答」页面提问，AI 会检索相关知识片段作答
5. **AI 摘要** → 知识详情页点击「AI 生成摘要」
6. **AI 标签** → 知识详情页点击「AI 推荐标签」

---

## 技术架构

```
前端 Vue 3 + Element Plus
    ↕ HTTP/SSE
后端 Spring Boot 3
    ├── Spring Security + JWT
    ├── MyBatis-Plus + MySQL 8（全文索引）
    ├── Redis（Token 黑名单）
    ├── Spring AI + DeepSeek（RAG 对话）
    └── Spring AI + Ollama（本地 Embedding）
            ↓
       ChromaDB（向量存储）
```
