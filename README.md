# Zhizhi
智知（Zhizhi）是一款基于Vue 3、Spring Boot 3与Spring AI技术栈开发的AI驱动型个人知识管理与智能问答系统。系统围绕“沉淀知识、智能检索、AI对话、知识图谱”四大核心理念设计，主要实现了用户登录模块、知识文档管理模块、AI智能问答模块自己是图谱模块四大功能模块。
